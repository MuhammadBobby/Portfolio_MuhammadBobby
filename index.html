#1 Project Web App & Dashboard Laundry

Website dan Dashboard Laundry adalah proyek yang dibuat oleh Muhammad Bobby & Team. Kami membangun Website & Dashboard Laundry untuk digunakan oleh Bisnis Jasa Laundry. Disini kami membangun Website sebagai antarmuka sistem yang berinteraksi dengan pelanggan dan Dashboard sebagai sistem manajemen yang disediakan untuk Pemilik, Karyawan, dll yang terintegrasi dengan antarmuka sistem. Dalam Dashboard juga terdapat leveling untuk masuk, dimana ada 3 level yang memiliki batasan yang berbeda, yaitu Pemilik, Admin, dan Karyawan.

Website dan Dashboard laundry terinstegrasi dengan database, dimana juga sudah dilengkapi Payment Gateway dengan berbagai macam metode pembayaran dan Notifikasi Realtime kepada pemilik/karyawan ketika pesanan layanan berhasil.

Bahasa & teknologi : HTML 5, Tailwind CSS, PHP, MySQL

Third Party : Midtrans (Payment Gateway) & Twilio API (Notifikasi Whatsapp Realtime)

#2 Project Dashboard Rental Playstation

Dashboard Rental Playstation ini merupakan sebuah Dashboard yang dibuat sebagai Dashboard yang bertujuan mengelola manajemen Rental Playstation. Tampilan Dashboard ini berupa tampilan admin, dimana terdapat tabel untuk menarik data hingga dapat menginput, mengedit, dan menghapus data, baik data Rental maupun transaksi Rental.

Dashboard ini menggunakan sistem CRUD yang terhubung dengan database local. Selain itu, Dashboard ini juga memiliki fitur pengeluaran laporan baik PDF maupun Excel. Ini tentunya akan menjadi fitur kunci dimana dashboard biasanya diakses oleh admin atau pimpinan yang memang membutuhkan fitur ini.

Bahasa & teknologi : HTML 5, Tailwind CSS, JavaScript, PHP, MySQL, Code Igniter 4

Third Party : DomPDF (export PDF) & PHPSpreadSheet (export Excel)
