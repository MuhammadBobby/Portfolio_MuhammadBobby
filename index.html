#2 Project Web Apotek

Web apotek ini adalah sebuah web yang dibuat sebagai web yang bertujuan mengelola manajemen barang masuk dan keluar di sebuah apotek yang dapat digunakan oleh admin dan owner. Tampilan web ini berupa tampilan admin, dimana terdapat tabel untuk menampilkan data hingga dapat menginput, mengedit, dan menghapus data, baik data barang maupun transaksi masuk dan keluar.
